20.有效的括号

发表于 2019-05-14 分类于 LeetCode ， 字符串

给定一个只包括 ‘(‘，’)’，’{‘，’}’，’[‘，’]’ 的字符串，判断字符串是否有效.

有效字符串需满足：

  • 左括号必须用相同类型的右括号闭合。
  • 左括号必须以正确的顺序闭合。

空字符串可被认为是有效字符串。

解答：

很显然可以用栈来做，但是python没有专门入栈的方法，需要用append().
这里直接用字符串替换，看最终是否剩下空串来判断括号是否有效
